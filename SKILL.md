#### 1.2 地缘政治事件

| 事件类型 | 影响板块 | 持续时间 | 典型案例 |
|---------|---------|---------|----------|
| **战争/冲突** | 军工、石油、黄金 | 1-6个月 | 俄乌冲突→军工、黄金 |

| **台海局势** | 全市场、航空 | 短期1-2周 | 情绪冲击为主 |

#### 1.3 突发事件

| 事件类型 | 影响特点 | 持续时间 | 典型案例 |
|---------|---------|---------|----------|
| **黑天鹅事件** | 急速下跌 | 1-3天 | 疫情爆发、瑞信危机 |

| **行业事故** | 行业龙头 | 1-4周 | 白酒塑化剂、医药集采 |
| **公司黑天鹅** | 单票闪崩 | 1-5天 | 财务造假、实控人被抓 |

#### 1.4 重大会议/时间窗口

| 时间节点 | 影响特点 | 交易策略 |
|---------|---------|----------|

| **中央经济工作会议** | 明年定调 | 春季行情预期 |
| **财报季** | 个股分化 | 业绩驱动为主 |

#### 1.5 大事件影响量化模型

```
事件冲击系数 = 事件级别 × 板块敏感度 × 时间衰减系数

事件级别（1-5分）：
- 5分：国家级重磅、战争、黑天鹅
- 4分：部委重要文件、行业政策
- 3分：地方政策、行业事故
- 2分：官方讲话、财报预警
- 1分：一般新闻、公司公告

板块敏感度（0.5-2.0）：
- 高敏感（1.5-2.0）：军工、科技、能源
- 中敏感（1.0-1.5）：消费、金融、制造
- 低敏感（0.5-1.0）：公用事业、医药

时间衰减系数：
- 第1天：1.0（冲击最大）
- 第3天：0.7
- 第7天：0.4
- 第14天：0.2
- 第30天：0.1
```

## 📊 收益准确率计算模型（V4.0新增）

### 2.1 历史收益追踪系统

| 追踪指标 | 计算方法 | 验证频率 |
|---------|---------|----------|
| **方向准确率** | 看涨/看跌正确次数/总预测次数 | 每日 |
| **目标价达成率** | 触及目标价的次数/总推荐次数 | 每周 |
| **止损触发率** | 止损触发的次数/总推荐次数 | 每周 |
| **收益率偏差** | 实际收益/预期收益 | 每月 |

### 2.2 准确率评分公式

```
综合准确率 = 方向准确率 × 40% + 目标价达成率 × 30% + 风险控制率 × 30%

其中：
- 方向准确率 = 正确预测次数 / 总预测次数
- 目标价达成率 = 触及目标价次数 / 推荐次数
- 风险控制率 = (止损未触发 + 亏损<5%) / 总交易

评级标准：
- 90-100%：⭐⭐⭐⭐⭐ 卓越
- 80-89%：⭐⭐⭐⭐ 优秀
- 70-79%：⭐⭐⭐ 良好
- 60-69%：⭐⭐ 一般
- <60%：⭐ 需改进
```

### 2.3 收益预期计算

| 评级 | 短线（<1月） | 中线（1-3月） | 长线（>3月） |
|-----|-------------|-------------|-------------|
| ⭐⭐⭐⭐⭐ | +15-25% | +25-50% | +50-100% |
| ⭐⭐⭐⭐ | +10-15% | +15-25% | +30-50% |
| ⭐⭐⭐ | +5-10% | +10-15% | +20-30% |
| ⭐⭐ | 0-5% | +5-10% | +10-20% |
| ⭐ | <0% | <5% | <10% |

### 2.4 胜率统计模型

```
预期收益 = Σ(概率 × 收益) - Σ(概率 × 亏损)

示例：
- 推荐上涨（70%概率涨15%，30%概率跌8%）
- 预期收益 = 0.7×15% - 0.3×8% = 10.5% - 2.4% = 8.1%

风控指标：
- 盈亏比 > 2:1 才推荐
- 最大亏损 < 10%
- 止损设置：-8%
- 止盈设置：+15-20%
```

## 📐 十维分析体系（完整版）

### 第0层：信息质量审计
- 数据来源可靠性验证（官方披露>三方数据）
- 财报真实性核查（勾稽关系、异常波动）
- 消息时效性确认（72小时内信息有效性）

### 第1层：宏观环境（水温判断）
- **政策水温**：货币政策、财政政策、行业监管政策
- **流动性水温**：M2增速、社融增量、北向资金净流入
- **情绪水温**：恐慌贪婪指数、融资融券余额、期权PCR

### 第2层：政策面分析
- **政策周期**：顶层设计→部委细则→地方落实
- **政策力度**：顶层文件>部委意见>地方规定
- **政策时滞**：政策出台到业绩兑现的时间窗口
- **政策受益度**：定量评估政策对公司的影响程度

### 第3层：行业逻辑（赛道选择）
- **终局推演**：行业天花板、市场空间、渗透率拐点
- **颠覆风险**：技术替代、政策冲击、行业格局变化
- **政策驱动**：行业催化剂、政策红利期
- **产业链位置**：上游资源/中游制造/下游品牌

### 第4层：公司基本面（核心层）
- **4M+模型**：
  - Model（商业模式）：盈利模式、现金流肖像
  - Moat（护城河）：壁垒类型（品牌/技术/资质/规模）
  - Management（管理层）：股权结构、高管背景、激励机制
  - Money（资金流）：经营现金流、投资现金流、筹资现金流
  - ESG（新增）：环境责任、社会责任、公司治理

### 第5层：竞争格局分析
- **波特五力**
- **竞争壁垒**：护城河深度与持续性
- **格局演变**：集中度变化趋势

### 第6层：技术面（辅助层）
- 均线系统、MACD、K线形态、筹码分布、量价关系

### 第7层：资金面（催化剂）
- 北向资金、机构持仓、游资动向、股东户数

### 第8层：行为金融层
- 机构动向、高管增减持、研报覆盖、股权激励

### 第9层：重大事件影响（V4.0新增）
- 政策大事件、地缘政治、突发事件、重大会议
- 事件冲击系数量化模型
- 时间窗口交易策略

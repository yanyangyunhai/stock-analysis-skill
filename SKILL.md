# 股票分析专家 - A股九维分析体系

## 📊 技能介绍

专业A股九维分析体系，提供全方位的股票分析服务。

### 核心功能

1. **政策面深度分析**
   - 政策周期评估
   - 政策力度量化
   - 政策受益度计算

2. **基本面全面扫描**
   - 4M+模型（Model, Moat, Management, Money）
   - 现金流肖像评估
   - 财务健康度检测

3. **技术面精准判断**
   - 均线系统
   - MACD+K线组合
   - 筹码分布分析

4. **资金面实时追踪**
   - 北向资金流向
   - 机构持仓变化
   - 股东集中度

5. **风险评估系统**
   - 七维度风险评分
   - 综合评级（85-100分=优质）

## 🎯 适用场景

- 个人投资者选股分析
- 机构投资决策辅助
- 股票研究报告生成
- 投资策略回测

## 💰 收益预期

- 准确率目标：>80%
- 风险收益比：>2:1
- 推荐仓位：<30%

## 📋 输出示例

```
股票代码：600519（贵州茅台）
评分：92分
评级：强烈推荐
目标价：2500元
止损位：2200元
理由：基本面优+品牌护城河+北向增持
风险：行业政策变化
```

## ⚠️ 风险提示

股市有风险，投资需谨慎。本技能仅供参考，不构成投资建议。

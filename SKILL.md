## 一，分析框架总览（2026年最新九维体系）

### 第0层：信息质量审计
- 数据来源可靠性验证（官方披露>三方数据）
- 财报真实性核查（勾稽关系、异常波动）
- 消息时效性确认（72小时内信息有效性）

### 第1层：宏观环境（水温判断）
- **政策水温**：货币政策、财政政策、行业监管政策
- **流动性水温**：M2增速、社融增量、北向资金净流入
- **情绪水温**：恐慌贪婪指数、融资融券余额、期权PCR

### 第2层：政策面分析（新增）
- **政策周期**：顶层设计→部委细则→地方落实
- **政策力度**：顶层文件>部委意见>地方规定
- **政策时滞**：政策出台到业绩兑现的时间窗口
- **政策受益度**：定量评估政策对公司的影响程度

### 第3层：行业逻辑（赛道选择）
- **终局推演**：行业天花板、市场空间、渗透率拐点
- **颠覆风险**：技术替代、政策冲击、行业格局变化
- **政策驱动**：行业催化剂、政策红利期
- **产业链位置**：上游资源/中游制造/下游品牌

### 第4层：公司基本面（核心层）
- **4M+模型**：
  - Model（商业模式）：盈利模式、现金流肖像
  - Moat（护城河）：壁垒类型（品牌/技术/资质/规模）
  - Management（管理层）：股权结构、高管背景、激励机制
  - Money（资金流）：经营现金流、投资现金流、筹资现金流
  - ESG（新增）：环境责任、社会责任、公司治理

### 第5层：竞争格局分析（新增）
- **波特五力**：
  - 供应商议价能力
  - 购买者议价能力
  - 新进入者威胁
  - 替代品威胁
  - 行业内竞争
- **竞争壁垒**：护城河深度与持续性
- **格局演变**：集中度变化趋势

### 第6层：技术面（辅助层）
- 均线系统：5/20/60/120/250日均线
- MACD：零轴金叉/死叉、柱状量、背离
- K线形态：锤子线、吞没、十字星
- 筹码分布：成本集中度、获利盘/套牢盘比例
- 量价关系：放量突破、缩量整理、天量见顶

### 第7层：资金面（催化剂）
- **配置型资金**：公募基金持仓、社保/险资动向
- **交易型资金**：游资炒作、杠杆资金、量化资金
- **北向资金**：沪股通/深股通净流入、行业偏好
- **股东结构**：国家队、机构、散户持仓比例

### 第8层：行为金融层
- 股东户数变化（筹码集中度）
- 高管增减持动态（窗口期效应）
- 研报覆盖密度（预期差）
- 股权激励/定增（稀释效应）

## 二，核心财务指标体系（升级版）

### 盈利能力指标
| 指标 | 优秀标准 | 说明 |
|-----|---------|------|
| ROE | >15% | 净资产收益率，越高越好 |
| ROIC | >10% | 投入资本回报率，更精准 |
| 毛利率 | >30% | 越高说明定价权越强 |
| 净利率 | >10% | 扣除所有费用后的利润率 |
| 扣非净利润增速 | >15% | 核心业务真实增长能力 |

### 估值指标（多维度）
| 指标 | 适用类型 | 评判标准 |
|-----|---------|---------|
| PE（市盈率） | 价值/周期股 | PE<20为低估，PE>40为高估 |
| PEG | 成长股 | PEG<1为低估，>1.5为高估 |
| PB（市净率） | 金融/重资产 | PB<2为低估 |
| PS（市销率） | 亏损股 | 越低越好 |
| EV/EBITDA | 周期/重资产 | <10为低估 |
| 股息率 | 价值股 | >3%为高股息 |

### 现金流肖像（新增）
| 类型 | 特征 | 评价 |
|-----|------|------|
| **奶牛型** | 经营现金流>0，投资现金流<0，筹资现金流<0 | ⭐⭐⭐⭐⭐ 优质 |
| **母鸡型** | 经营现金流>0，投资现金流<0，筹资现金流=0 | ⭐⭐⭐⭐ 良好 |
| **蛮牛型** | 经营现金流>0，投资现金流>>0，筹资现金流>0 | ⭐⭐⭐ 扩张期 |
| **赌徒型** | 经营现金流<0，投资现金流<0，筹资现金流>0 | ⭐⭐ 高风险 |
| **失血型** | 经营现金流<0，投资现金流>0，筹资现金流<0 | ⭐ 差 |

### 财务健康指标
| 指标 | 安全标准 | 说明 |
|-----|---------|------|
| 资产负债率 | <60% | 越低越安全 |
| 流动比率 | >1.5 | 短期偿债能力 |
| 经营现金流/净利润 | >1 | 利润含金量 |
|应收账款增速 | <营收增速 | 回款能力 |
| 商誉/净资产 | <20% | 商誉占比过高有爆雷风险 |

## 三，政策面分析框架（新增）

### 政策评估矩阵
| 政策类型 | 评估维度 | 权重 |
|---------|---------|------|
| 顶层政策 | 战略意义、推进力度、资金保障 | 40% |
| 部委细则 | 操作性、执行时限、配套资金 | 30% |
| 地方落实 | 属地责任、考核压力、地方财力 | 30% |

### 政策受益度评估
```
政策受益度 = 政策力度系数 × 公司业务契合度 × 市值弹性系数

政策力度系数：
- 国家级规划：1.0
- 部委重要文件：0.8
- 部委意见/函：0.6
- 地方政策：0.4

业务契合度：
- 主业直接受益：1.0
- 间接受益（产业链）：0.6
- 边缘受益：0.3

市值弹性系数：
- 小市值（<50亿）：1.5
- 中市值（50-200亿）：1.2
- 大市值（>200亿）：1.0
```

## 四，技术分析核心指标（升级版）

### 趋势判断
1. **均线多头排列**：5>20>60>120>250日均线
2. **MACD零轴上金叉**：趋势向好，柱状量放大确认
3. **成交量放大**：量能配合是关键
4. **筹码低位集中**：主力吸筹阶段

### 买卖信号（量化）
| 信号类型 | 技术指标组合 | 强度 |
|---------|------------|------|
| 强烈买入 | MACD金叉 + 放量突破 + 均线多头 + 筹码低位集中 | ⭐⭐⭐⭐⭐ |
| 买入信号 | MACD零轴上金叉 + 缩量回踩均线不破 | ⭐⭐⭐⭐ |
| 观望信号 | 均线纠缠 + 缩量 + 筹码分散 | ⭐⭐ |
| 卖出信号 | MACD死叉 + 缩量 + 均线空头排列 | ⭐⭐⭐⭐ |
| 强烈卖出 | 高位放量滞涨 + 顶背离 + 筹码高位分散 | ⭐⭐⭐⭐⭐ |

### 风险信号（量化）
- **顶背离**：股价创新高，MACD不创新高 → 顶部信号
- **天量见顶**：成交量创历史新高 → 主力出货
- **高位孕线**：高位出现十字星 → 多空分歧
- **破位下跌**：股价跌破60日均线 → 中期趋势破坏

## 五，资金面分析（升级版）

### 北向资金追踪（新增）
| 指标 | 计算方式 | 信号含义 |
|-----|---------|---------|
| 持股变动 | 本期持股-上期持股 | 正=增持，负=减持 |
| 持仓占比 | 持股/总股本 | 持续提升=长线看好 |
| 资金流向 | 净买入额 | 持续流入=看好 |
| 行业偏好 | 配置比例变化 | 反映外资行业判断 |

### 机构持仓分析
```
机构持仓变化 = 本期机构持仓比例 - 上期机构持仓比例

> +5%：大幅增仓（强烈看好）
> +2%：增仓（看好）
-2%~+2%：中性
< -2%：减仓（看空）
< -5%：大幅减仓（强烈看空）
```

### 筹码分布分析（新增）
| 筹码形态 | 技术含义 |
|---------|---------|
| 单峰密集 | 主力控盘度高，拉升在即 |
| 双峰峡谷 | 压力/支撑明显，震荡整理 |
| 低位密集 | 主力吸筹，后市看涨 |
| 高位密集 | 主力出货，后市看跌 |

## 九，估值方法论（新增）

### 多维度估值对比
| 估值方法 | 适用场景 | 优缺点 |
|---------|---------|-------|
| PE估值 | 稳定增长企业 | 简单直观，但受行业周期影响 |
| PEG估值 | 成长型企业 | 结合增速，更合理 |
| PB估值 | 重资产/金融企业 | 账面价值为底 |
| DCF估值 | 现金流稳定企业 | 理论上最准确，但假设多 |
| EV/EBITDA | 周期/并购企业 | 剔除资本结构影响 |
| 股息率估值 | 价值/分红型 | 适用于保守投资者 |

### 相对价值评估
```
相对价值 = 公司PE / 行业中位数PE
相对价值 < 0.8：相对低估
相对价值 0.8-1.2：相对合理
相对价值 > 1.2：相对高估
```

## 十二，准确率追踪系统

### 追踪指标
| 指标类型 | 追踪内容 | 验证方法 |
|---------|---------|---------|
| 方向判断 | 看涨/看跌/震荡 | 次日涨跌验证 |
| 目标价 | 预测价格区间 | 是否触及 |
| 支撑压力位 | 关键价格点位 | 价格是否触及 |
| 止损位 | 预设止损价 | 是否触发 |
| 风险预警 | 预警风险点 | 是否发生 |

### 准确率计算
```
准确率 = 准确预测次数 / 总预测次数 × 100%

目标：
- 第一阶段（1-10天）：准确率>70%
- 第二阶段（11-30天）：准确率>80%
- 第三阶段（30天+）：准确率>85%
```
